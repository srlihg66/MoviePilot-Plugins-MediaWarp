> **为 MoviePilot 快速接入 MediaWarp 的第三方插件**
> 
> 让你在 MoviePilot 中一键部署并管理 Emby / Jellyfin / 飞牛影视 的增强反代中间件！

## 🌟 项目简介

### 本插件 (MediaWarp Plus) 的作用
通过安装此插件，你可以直接在 MoviePilot 的插件市场中：
- **一键安装** MediaWarp，免去繁琐的手动部署环境步骤。
- **可视化配置**，直接在 MoviePilot 前端页面集中管理 MediaWarp 的各项运行参数。

### MediaWarp 原项目核心亮点
MediaWarp 是一款强大的媒体网关中间件插件，专门用于优化原媒体服务器的播放体验：
- 🚀 **Strm 直链播放**：支持 HTTPStrm / AlistStrm 文件 302 重定向直链播放，彻底解放你的家庭宽带上行带宽！
- 🎨 **前端美化增强**：支持注入自定义 HTML/CSS/JS，原生适配 emby-crx 等美化包。
- 🛡️ **客户端访问控制**：轻松过滤和屏蔽特定设备或播放客户端。
- 🔧 **丰富的嵌入脚本**：包括外部播放器调用（Emby）、Web端弹幕、同人图展示（Fanart）等实用组件。

## 🛠️ 如何安装 / 添加插件

本仓库目前专注于提供 **MediaWarp Plus** 单一插件，安装过程非常简单：

### 第一步：添加插件市场仓库
在 MoviePilot 的插件市场设置中，添加本仓库的 URL 以订阅插件：
- **仓库地址**：`https://github.com/srlihg66/MoviePilot-Plugins-MediaWarp`

### 第二步：安装并启用
1. 添加完成后，点击**刷新**插件市场。
2. 在列表中找到 **MediaWarp Plus** 插件并点击安装。
3. 安装成功后，根据插件说明配置相关参数并启用，即可享受极致体验！
